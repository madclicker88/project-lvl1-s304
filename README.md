Games for CLI

List of games:
* brain-even
* brain-calc
* brain-gcd
* brain-balance
* brain-progression
* brain-prime

### Install
```
npm install -g brain-games-madclicker88
```

### Usage
To launch the game, enter the name of the game in CLI, e.g.:
```
brain-progression
```
